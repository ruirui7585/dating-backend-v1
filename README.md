# Female Operator Backend System

Backend configuration and rule system for the Female Operator App.

Decoupled from the frontend prototype. Defines model profiles, traffic allocation, scheduling, risk controls, commission/payout rules, and system-level constraints.

## Directory

- `backend-config/`: Model configuration, traffic rules, scheduling rules, risk rules, commission rules, system constraints
- `config-docs/`: Configuration documentation and rule explanations
- `rules/`: Backend rule definitions and validation logic
- `api-schemas/`: API schema definitions for backend endpoints
- `archive/`: Archived or deprecated backend configurations
- `backend-config.json`: Backend system metadata and scripts
